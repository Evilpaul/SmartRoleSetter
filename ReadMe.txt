SmartRoleSetter
Created by Aeldra (EU-Proudmoore)
***********************************************************************

Sets automatically your role in the raid based on your skill tree.


FAQ:
Q1: How can I configure it?
A1: Use the chat command /srs ? for help


Features:
* Supports all languages


Chat:
Type /srs or /srole to toggle SmartRoleSetter On/Off
Type /srs [command] or /srole [command] in game
?    - Help
auto - Toggles auto accept role poll
